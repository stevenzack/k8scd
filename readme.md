# K8sCD

Simple GitOps tool for Kubernetes

# Requirements

- Install `Git`
- Install `kubectl`

Install k8scd

```
mkdir ~/.k8scd
curl https://github.com/stevenzack/k8scd/releases/download/latest/k8scd_linux_amd64-latest -o ~/.k8scd/k8scd
cd ~/.k8scd && ./k8scd &
```

And open K8sCD web UI in [http://localhost:9876](http://localhost:9876)
